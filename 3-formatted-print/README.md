# Formatted Print

Printing macros in `std::fmt`:
- `format!`: Creates and returns a formatted String without printing anything to the console.
- `print!`: Prints formatted text to standard output (stdout) without adding a trailing newline.
- `println!`: Prints formatted text to standard output (stdout) and appends a newline at the end.
- `eprint!`: Prints formatted text to standard error (stderr) without adding a trailing newline.
- `eprintln!`: Prints formatted text to standard error (stderr) and appends a newline at the end.


[Example](./formatted-print.rs)
```rust
fn main() {
    println!()
}
```
